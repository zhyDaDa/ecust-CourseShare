### 1.1 集合


#### 1.1.1 基本定义


- 集合是具有相同性质的一组事物的整体。
- 元素用 `x ∈ X` 表示属于集合 `X`。
- 常见表示方法：
  - 列举法
  - 描述法
- 空集合必须用 `set()`，不能用 `{}`，因为 `{}` 在 Python 中表示空字典。

#### 1.1.2 集合的特点


- 无序性：元素之间没有先后顺序。
- 确定性：元素要么属于集合，要么不属于。
- 互异性：集合中的元素不重复。

#### 1.1.3 常见运算


- 并集 `A ∪ B`
- 交集 `A ∩ B`
- 差集 `A \ B`
- 补集
- 幂集 `P(A)`

#### 1.1.4 集合的性质

##### 1.1.4.1 集合关系补充

- **集合等势（Equivalence）**：如果集合 A 中的元素和集合 B 中的元素存在一一对应关系（双射），则称 A 与 B 等势，记作 A ≈ B。例如，整数集 ℤ ≈ 自然数集 ℕ（存在双射 f: ℤ → ℕ）。
- **集合的优势**：若集合 A 中每个元素在集合 B 中都有它独有的相对应元素（单射），则称 B 优势于 A，记作 A ⪯ B。若 A ⪯ B 且 A ≉ B，则称 B 真优势于 A。
- **有限集与无限集**：一个集合是有限的当且仅当它与某个自然数等势；否则称为无限集。最小的无限集合是自然数集，无穷集合和它的真子集有相同的基数。
- **可数集与不可数集**：可数集合是与自然数集的某个子集具有相同基数（等势）的集合。任何势大于自然数集的集合称为不可数集合。实数集、无理数集都是无限不可数集。

##### 1.1.4.2 笛卡尔积的定义与性质

- **笛卡尔积（Cartesian Product）**：又称直积，是两个集合 A 与 B 所有可能的有序对组成的集合：
  - A × B = {(a, b) | a ∈ A ∧ b ∈ B}
- 性质：
  - A × ∅ = ∅ × A = ∅
  - 一般不满足交换律和结合律，对集合的并和交满足分配律
  - A × (B ∪ C) = (A × B) ∪ (A × C)
  - A × (B ∩ C) = (A × B) ∩ (A × C)
  - A × B ∩ C × D = (A ∩ C) × (B × D)
  - 若 A 包含无限多元素，则 A × A 有和 A 一样多的元素。

##### 1.1.4.3 集合族、幂集与康托尔定理

- **集合族（Family of Sets）**：由具有某种性质的一些集合所构成的集合，即"集合的集合"。
- **幂集（Power Set）**：对于有限个元素构成的集合 A，由该集合的所有子集构成的集合称为该集合 A 的幂集，记作 𝒫(A) ≔ {S | S ⊆ A}。若 |A| = n，则 |𝒫(A)| = 2ⁿ。
- **康托尔定理**：任何集合 A 的幂集 𝒫(A) 的势严格大于集合 A 的势；可数无限集合构造的幂集的基数是不可数无限。

##### 1.1.4.4 σ-代数（σ-Algebra）的定义与性质

- **定义**：某个集合 Ω 上的 σ-代数（又称 σ-域）是 Ω 的所有子集的集合（即幂集 𝒫(Ω)）的一个子集合 ℱ ⊆ 𝒫(Ω)，满足以下三个条件：
  ① Ω 和 ∅ 是 ℱ 的元素，即 ∅ ∈ ℱ，Ω ∈ ℱ
  ② 如果集合 B ∈ ℱ，那么 B'（补集）∈ ℱ
  ③ 如果有可数个集合 B₁, B₂, ... ∈ ℱ，那么 ∪ᵢ Bᵢ ∈ ℱ
- **最小与最大 σ-代数**：{∅, Ω} 为 Ω 上含元素最少的 σ-代数，𝒫(Ω) 为含最多元素的 σ-代数。
- σ-代数满足可数个集合的并集、交集和补集运算的封闭性。

##### 1.1.4.5 可测空间与测度

- **可测空间（Measurable Space）**：称 (Ω, ℱ) 为一个可测空间，其中 Ω 为非空集合，ℱ 为 Ω 上的一个 σ-代数，ℱ 中的元素 S（S ⊆ Ω）称为可测集合。
- **测度（Measure）**：对于可测空间 (Ω, ℱ)，测度 μ : ℱ → [0, ∞) 是定义在 ℱ 上的非负函数，满足 μ(∅) = 0 和可数可加性。三元组 (Ω, ℱ, μ) 称为一个测度空间。
- **概率空间（Probability Space）**：(Ω, ℱ, P) 中：
  - Ω — 样本空间，试验中所有可能结果的集合
  - ℱ — 事件集合，是 Ω 的一些子集构成的集合，满足 σ-代数的属性
  - P — 概率测度，P : ℱ → [0, 1]，满足 P(Ω) = 1 和可数可加性

##### 1.1.4.6 上确界与下确界

- **上确界/最小上界（Supremum）**：对于 ∅ ≠ S ⊆ ℝ，若 ∀s ∈ S 均有 s ≤ t，且 t 是满足此不等式的最小值，则 t 为 S 的上确界，记作 sup S。若不存在则称上确界为 +∞。
- **下确界/最大下界（Infimum）**：对于 ∅ ≠ S ⊆ ℝ，若 ∀s ∈ S 均有 t ≤ s，且 t 是满足此不等式的最大值，则 t 为 S 的下确界，记作 inf S。
- **对偶性**：inf S = −sup(−S)
- **与最值的区别**：上/下确界不一定在集合中取到。例如 inf {x ∈ ℝ : x > 2} = 2，但 2 不在该集合中。

##### 1.1.4.7 凸集与凸包

- **凸集（Convex Set）**：若集合 C 中任意两点的连线都在 C 内，则称 C 为凸集，即 ∀x, y ∈ C，∀λ ∈ [0, 1]，有 λx + (1−λ)y ∈ C。
- **凸包（Convex Hull）**：包含 X 的最小凸集称为 X 的凸包。∀X ⊆ ℝⁿ，包含 X 的最小凸集便是 X 的凸包。

#### 1.1.5 模糊数学


> 这一部分属于 `1-1 集合` 的延伸内容，不是独立章节。

##### 1.1.5.1 基本思想

- 模糊数学处理“属于某集合的程度”而不是简单的“属于/不属于”。
- 通过隶属函数 `μ_A(x)` 把元素映射到 `[0,1]`。

##### 1.1.5.2 表示方法

- **解析法**：给出隶属函数的具体表达式
- **Zadeh（泽德）记法**：
  - 有限集或可列集：A = μ_A(x₁)/x₁ + μ_A(x₂)/x₂ + ...（"∑"和"+"表示各元素与隶属度对应关系的总括，非数学求和）
  - 连续集或无限集：A = ∫ μ_A(x)/x（"∫"表示各元素与隶属度对应关系的一个总括）
- **向量法（模糊向量）**：A = [μ_A(x₁), μ_A(x₂), ..., μ_A(xₙ)]，隶属度为 0 的项不能省略，元素次序不能随意调换
- **序偶法**：A = {(x₁, μ_A(x₁)), (x₂, μ_A(x₂)), ..., (xₙ, μ_A(xₙ))}
- **图解法**：文氏图、超立方体上点的集合

##### 1.1.5.3 截集与清晰化

- **λ-截集的定义**：设 X 是论域，A ∈ ℱ(X)，λ ∈ [0,1]，令
  - A_λ ≔ {x ∈ X | μ_A(x) ≥ λ}（λ 截集）
  - A_λ̅ ≔ {x ∈ X | μ_A(x) > λ}（λ 强截集）
  - λ 称为阈值或置信水平
- 模糊集 A 的 λ 截集与 λ 强截集**均为经典集合**
- **截集的性质**（设 A, B ∈ ℱ(X)，λ ∈ [0,1]）：
  - (A ∩ B)_λ = A_λ ∩ B_λ，(A ∪ B)_λ = A_λ ∪ B_λ
  - A_λ̅ ⊆ A_λ
  - λ ≤ η ⟹ A_η ⊆ A_λ
  - (A_λ)_μ = A_(λ∨μ)，(A_λ)_μ = A_(λ∧μ)
- **截积/数积**：对于 λ ∈ [0,1]，A ∈ ℱ(X)，定义 λA 为 λ 与 A 的数积（截积），其隶属函数为 μ_λA(x) = min{λ, μ_A(x)}，∀x ∈ X。λA 仍为 X 上的模糊集合。

##### 1.1.5.4 核、边界、支集

- **核（Ker）**：Ker A = {x ∈ X | μ_A(x) = 1}
  - 若 Ker A ≠ ∅，称 A 为正规模糊集；否则为非正规模糊集
- **边界（Bound）**：Bound A = {x ∈ X | 0 < μ_A(x) < 1}
- **支集（Supp）**：Supp A = {x ∈ X | μ_A(x) > 0}
  - Supp A = Ker A ∪ Bound A
- **性质**：
  - A₁ = Ker A，Supp(Supp A) = Supp A
  - A ⊆ B ⟹ Ker A ⊆ Ker B，Supp A ⊆ Supp B
  - Ker(A ∪ B) = Ker A ∪ Ker B，Supp(A ∪ B) = Supp A ∪ Supp B
  - Ker(A ∩ B) = Ker A ∩ Ker B，Supp(A ∩ B) = Supp A ∩ Supp B
  - Ker(A') = (Supp A)'，Supp(A') = (Ker A)'
  - **分界点 x₀**：μ_A(x₀) = 0.5
- **凸模糊集合**：对于 ∀x₁, x₂ ∈ X，∀λ ∈ [0,1]，有 μ_A(λx₁ + (1−λ)x₂) ≥ min{μ_A(x₁), μ_A(x₂)}

##### 1.1.5.5 分解定理

- 模糊集可以由一族截集加权“拼回去”。
- 这说明模糊集和经典集合之间可以相互转换。

### 1.3 函数


#### 1.3.1 基本性质


- 定义域、值域、单调性、奇偶性、周期性。
- 复合函数、反函数、分段函数。
- 函数图像的理解是后续连续性、导数和积分的基础。

#### 1.3.2 函数的阶估计（无穷小量）

- **无穷小量**：若 lim_{x→x₀} f(x) = 0，则称 f(x) 是 x → x₀ 时的无穷小量，记作 f(x) = o(1)
- **无穷小量的分类**（设 f 和 g 都是 x → x₀ 的无穷小量）：
  - 高阶无穷小：lim f(x)/g(x) = 0，记作 f(x) = o(g(x))
  - 低阶无穷小：lim f(x)/g(x) = ∞
  - 同阶无穷小：lim f(x)/g(x) = c ≠ 0
  - 等价无穷小：lim f(x)/g(x) = 1，记作 f(x) ~ g(x)
- **等价替换定理**：若 x → x₀ 时 f(x) ~ g(x)，则在乘除运算中等价无穷小可以互相替换而不影响结果
- **常见等价无穷小**（x → 0 时）：
  - sin x ~ x，tan x ~ x，arcsin x ~ x
  - ln(1+x) ~ x，eˣ − 1 ~ x
  - 1 − cos x ~ x²/2
  - (1+x)ᵃ − 1 ~ ax
  - √(1+x) − 1 ~ x/2，∛(1+x) − 1 ~ x/3

#### 1.3.2 函数在 AI 中的意义


- 激活函数、损失函数、本质上都是函数。
- 机器学习模型通常是在函数空间里做优化。

### 1.4 导数与微分


#### 1.4.1 导数的定义


- 导数刻画函数在某点的瞬时变化率。
- 几何上对应切线斜率。
- 物理上常表示速度、变化率。

#### 1.4.2 基本求导规则


- 和、差、积、商法则
- 链式法则
- 隐函数求导
- 参数方程求导
- 高阶导数

#### 1.4.3 微分


- 微分是函数在一点附近的线性近似。
- 全微分用于多元函数的一阶近似。
- **极值点与驻点**：若 f'(x0) = 0，则 x0 为驻点；若 f''(x0) > 0 则为局部极小值点，f''(x0) < 0 则为局部极大值点

#### 1.4.4 梯度下降法（综合案例）


- **梯度下降法原理**：已知可微函数 f(x) 的初始值 x0 和迭代步长 alpha（学习率），在 x0 的梯度为 grad f(x0)，则下一个点的坐标为 x1 = x0 - alpha * grad f(x0)，反复迭代直到收敛
- **步长选择**：alpha 过小收敛较慢，alpha 过大迭代会出现震荡或不收敛
- **初值问题**：若 f(x) 为凸函数则收敛到全局最小值；若非凸则只能收敛到局部最小值
- **示例**：f(x) = x^2，x0=1，alpha=0.4，x1=0.2，x4=0.00016
- **Python 实现**：使用 Matplotlib 和 mpl_toolkits.mplot3d 绘制三维函数图像，通过迭代求最小值

### 1.5 积分学


#### 1.5.1 不定积分


- **定义**：若存在可导函数 F(x) 使得 F'(x) = f(x)，则 F(x) 是 f(x) 的一个原函数，F(x) + C 为 f(x) 的全体原函数。int f(x)dx = F(x) + C
- **换元积分法**：凑微分法 int f(u(x))u'(x)dx = int f(u)du；变量替换法
- **分部积分法**：int u(x)v'(x)dx = u(x)v(x) - int u'(x)v(x)dx

#### 1.5.2 定积分


- **定义**：int_a^b f(x) dx = lim_{Delta x->0} sum f(xi_i) Delta x_i
- **牛顿-莱布尼茨公式**：int_a^b f(x) dx = F(b) - F(a)，其中 F'(x) = f(x)
- **几何意义**：将区间 [a,b] 无限细分，间隔为 dx，函数增量变为微分 dy = f'(x)dx，因此 f(b)-f(a) = sum dy = int f'(x)dx
- **积分中值定理**：int_a^b f(x)dx = f(xi)(b-a)，其中 xi in [a,b]

#### 1.5.3 常用积分技巧


- 换元积分
- 分部积分
- 常见函数积分

#### 1.5.4 Python 求解积分


- **SymPy 求定积分**：integrate(fx, (x, a, b))
- **数值积分函数**：quad（单积分）、dblquad（二重积分）、tplquad（三重积分）、trapz（梯形规则）、simps（辛普森规则）
- **多元积分**：先对一个变量积分，再对另一个变量积分

### 1.6 极限与连续


#### 1.6.1 极限的基本思想


- 极限描述变量在趋近过程中的稳定趋势。
- 计算极限时常用：
  - 夹逼
  - 等价无穷小
  - 洛必达法则
  - 泰勒展开

#### 1.6.2 连续性


- 连续的直观含义是图像“没有断点”。
- 连续函数在闭区间上有很多良好性质：
  - 有界性
  - 最值存在
  - 介值性

#### 1.6.3 一致连续


- 一致连续比普通连续更强。
- 定义中 `δ` 只依赖 `ε`，不依赖具体点的位置。
- 课程中强调：
  - 连续不一定一致连续。
  - 闭区间上的连续函数必一致连续。
  - `1/x` 在 `(0,+∞)` 上不是一致连续，但在 `[1,+∞)` 上是一致连续。

#### 1.6.4 Lipschitz 连续


- 如果存在常数 `L`，使得 `|f(x)-f(y)| ≤ L|x-y|`，则称为 Lipschitz 连续。
- Lipschitz 连续通常比一致连续更强。
- 作业中专门要求判断 Lipschitz 性并计算常数。

### 1.7 多元函数


#### 1.7.1 偏导数与全微分


- 偏导数刻画一个变量变化时函数的变化率。
- 全微分是多元函数的线性近似。

#### 1.7.2 梯度与方向导数


- 梯度指向函数增长最快的方向。
- 方向导数刻画沿某个方向的变化率。
- 课程作业中要求计算梯度并沿给定方向求方向导数。

### 1.8 泰勒展开


- 泰勒展开用多项式近似复杂函数。
- 常用于：
  - 局部近似
  - 极限计算
  - 误差分析

### 1.9 常微分方程


- **基本概念**：含有自变量、未知一元函数 y=y(x) 以及函数各阶导数的方程称为常微分方程（ODE），方程的阶数为最高阶导数的阶数
- **线性微分方程**：a_n(x)y^(n) + ... + a_0(x)y = g(x)；否则为非线性。若系数均为常数则为常系数线性微分方程
- **齐次与非齐次**：g(x)=0 为齐次，否则为非齐次
- **偏微分方程（PDE）**：含有未知多元函数及其偏导数的微分方程
- **通解与特解**：包含 n 个独立任意常数的解为通解，不含任意常数的解为特解
- **初值问题——柯西-利普希茨定理**：dy/dx = f(x,y)，y(x0)=y0，若 f 在 (x0,y0) 附近连续且对 y 是 Lipschitz 连续的，则解存在唯一
- **一阶线性微分方程的求解（积分因子法）**：
  - 齐次：dy/dx + ay = 0 => y = Ce^{-ax}
  - 非齐次：dy/dx + ay = b(x) => y = e^{-ax} int e^{ax} b(x) dx + C
  - 一般形式：dy/dx + a(x)y = b(x) => 积分因子 mu = e^{int a(x)dx}
- **数值方法**：龙格-库塔（Runge-Kutta）法和理查森法
- **Python 求解**：使用 SymPy 中的 dsolve 函数


## 2. 线性代数

### 2.1 向量


#### 2.1.1 向量的定义


- 向量由若干分量组成，既有大小又有方向。
- 零向量长度为 0。
- 单位向量长度为 1。

#### 2.1.2 向量与机器学习


- 样本通常可写成特征向量。
- 向量是数据表示的最基本形式。

#### 2.1.3 向量的核心运算


- 加法、数乘
- 内积

- 投影

### 2.2 范数与距离


#### 2.2.1 范数


- 范数用于度量向量大小。
- 必须满足：
  - 非负性
  - 齐次性
  - 三角不等式

#### 2.2.2 常见距离


- 欧氏距离
- 曼哈顿距离
- 切比雪夫距离


### 2.3 矩阵


#### 2.3.1 基本概念


- 矩阵是按行列排列的数表。
- 本质上可看作线性变换的表示。

#### 2.3.2 基本运算


- 加法、数乘、乘法
- 转置
- 逆矩阵
- 迹

#### 2.3.3 矩阵在 AI 中的作用


- 数据批处理
- 网络权重表示
- 图结构表示
- 协方差矩阵、拉普拉斯矩阵

### 2.4 行列式


- **定义**：n 阶方阵 A 的行列式 det(A) 是一个标量，等于所有不同行不同列元素乘积的代数和。
- **性质**：|A^T| = |A|；两行互换变号；某行乘以 k，行列式乘以 k；行列式等于所有特征值的乘积
- 行列式为 0 常意味着线性相关或不可逆，此时至少有一个特征值为 0
- **代数余子式展开（拉普拉斯展开）**：行列式可按任意一行或一列展开

### 2.5 线性方程组


- **高斯消元法**：将增广矩阵通过行初等变换化为行阶梯形矩阵，然后回代求解
- **解的判定**：rank(A)=rank([A|b])=n 有唯一解；rank(A)=rank([A|b])<n 有无穷多解；rank(A)<rank([A|b]) 无解
- **齐次方程组 Ax=0**：解空间维数 = n-rank(A)，基础解系为解空间的一组基
- **非齐次方程组 Ax=b**：通解 = 齐次通解 + 非齐次特解

### 2.6 线性相关、秩与基


- 线性相关/无关决定空间维数。
- 极大线性无关组是构造基的基础。
- 秩是矩阵最重要的结构量之一。

### 2.7 正交化与正交基


- Gram-Schmidt 正交化用于把线性无关向量组转成标准正交基。
- 正交基有助于简化投影和分解问题。

### 2.8 二次型


- 二次型是形如 `x^T A x` 的表达式。
- 判断正定性是二次型的重要问题。
- 正定、半正定、负定、半负定常与特征值符号对应。

### 2.9 特征值、特征向量与分解


#### 2.9.1 基本概念


- 若 `Ax = λx`，则 `λ` 为特征值，`x` 为特征向量。
- 特征值揭示矩阵的固有变换性质。

#### 2.9.2 代数重数与几何重数


- 代数重数：特征值作为根出现的次数。
- 几何重数：对应特征空间维数。

#### 2.9.3 矩阵分解


- **特征分解（对角化）**：A = P * Lambda * P^{-1}，其中 Lambda 对⻆元为特征值，P 的列向量为对应特征向量
- **奇异值分解（SVD）**：A = U * Sigma * V^T，U、V 为正交矩阵，Sigma 的对角元为奇异值
  - **几何意义**：SVD 将线性变换分解为旋转（V^T）、缩放（Sigma）和旋转（U）的组合
  - **图像压缩**：保留最大的 k 个奇异值可近似重构图像
  - **推荐系统**：将用户-物品评分矩阵分解为低秩矩阵，预测缺失评分
- **瑞利商**：R(A,x) = (x^T A x)/(x^T x)，最大值和最小值分别对应最大和最小特征值
  - **广义瑞利商**：R(A,B,x) = (x^T A x)/(x^T B x)，B 正定
  - **PCA 应用**：最大投影方差方向对应最大特征值特征向量
  - **LDA 应用**：最大化类间/类内散度比率的广义瑞利商
  - 正交分解
  - 特征分解



## 3. 概率与数理统计

来源：`概率与数理统计.pdf`

### 3.1 概率论的基本概念

#### 3.1.1 随机试验

- 随机现象：结果不确定的现象（如彩票中奖、天气状况）
- 必然现象：结果确定的现象（如上抛物体掉落到地上）
- 随机试验的三大特性：
  - 可以在相同条件下重复进行
  - 事先知道可能出现的结果
  - 进行试验前不知道哪个结果会发生

#### 3.1.2 样本空间与随机事件

- 样本空间 S：随机试验 E 的所有可能结果构成的集合，S 中的元素 e 称为样本点
- 随机事件：S 的子集 A 称为 E 的随机事件 A，当且仅当 A 所包含的一个样本点发生称事件 A 发生
- **事件的关系及运算**：
  - 包含：A ⊆ B（A 发生则 B 一定发生）
  - 相等：A = B
  - 并（和）：A ∪ B（A 或 B 至少一个发生）
  - 交（积）：A ∩ B（A 和 B 同时发生）
  - 差：A \ B（A 发生而 B 不发生）
  - 互斥（互不相容）：A ∩ B = ∅
  - 对立事件（逆事件）：A'
- **事件运算律**：
  - 交换律：A ∪ B = B ∪ A，A ∩ B = B ∩ A
  - 结合律：(A ∪ B) ∪ C = A ∪ (B ∪ C)，(A ∩ B) ∩ C = A ∩ (B ∩ C)
  - 分配律：A ∩ (B ∪ C) = (A ∩ B) ∪ (A ∩ C)，A ∪ (B ∩ C) = (A ∪ B) ∩ (A ∪ C)
  - 德摩根律：(A ∪ B)' = A' ∩ B'，(A ∩ B)' = A' ∪ B'
  - 对偶律、幂等律、吸收律

#### 3.1.3 频率与概率

- 频率：n 次重复试验中事件 A 出现的次数 n_A 与 n 的比值 f_n(A) = n_A / n
- **概率的统计定义**：当 n 充分大时，频率稳定在某个常数 p 附近，p 称为 A 的概率，记作 P(A)
- **概率的公理化定义**：
  - 非负性：0 ≤ P(A) ≤ 1
  - 规范性：P(S) = 1
  - 可列可加性：对两两互不相容的事件 A₁, A₂, ...，有 P(∪Aᵢ) = Σ P(Aᵢ)
- **概率的性质**：
  - P(∅) = 0
  - P(A') = 1 − P(A)
  - 加法公式：P(A ∪ B) = P(A) + P(B) − P(A ∩ B)
- **古典概型**：样本空间有限且每个样本点等可能，P(A) = |A| / |S|
- **几何概型**：样本空间为某区域（长度/面积/体积），P(A) = A 的度量 / S 的度量

#### 3.1.4 条件概率与独立性

- **条件概率**：P(B|A) = P(AB) / P(A)，P(A) > 0，表示 A 发生的条件下 B 发生的概率
- **乘法公式**：P(AB) = P(A) · P(B|A) = P(B) · P(A|B)
- **全概率公式**：若 B₁, B₂, ..., Bₙ 为完备事件组（两两互不相容且 ∪Bᵢ = S），则：
  P(A) = Σ_{i=1}^{n} P(Bᵢ) · P(A|Bᵢ)
- **贝叶斯公式**：P(Bᵢ|A) = P(Bᵢ) · P(A|Bᵢ) / Σ_{j=1}^{n} P(Bⱼ) · P(A|Bⱼ)
  先验概率 P(Bᵢ) -> 后验概率 P(Bᵢ|A)
- **独立性定义**：P(AB) = P(A)·P(B)，则 A 与 B 相互独立
- 若 P(A) > 0，则 A 与 B 独立等价于 P(B|A) = P(B)

### 3.2 随机变量及其分布

#### 3.2.1 随机变量

- 随机变量 X 将随机试验的结果映射到实数，是定义在样本空间 S 上的实值函数

#### 3.2.2 分布描述

- **离散型随机变量**：概率质量函数 P(X = xᵢ) = pᵢ，满足 pᵢ ≥ 0 且 Σpᵢ = 1
- **连续型随机变量**：概率密度函数 f(x)，满足 f(x) ≥ 0 且 ∫f(x)dx = 1
- **分布函数**：F(x) = P(X ≤ x)，是统一描述工具，单调非减、右连续、F(−∞)=0、F(+∞)=1

#### 3.2.3 常见离散分布

- **伯努利分布（0-1 分布）**：P(X=1) = p，P(X=0) = 1−p，E(X)=p，D(X)=p(1−p)
- **二项分布**：X ∼ B(n, p)，P(X=k) = C(n,k)p^k(1−p)^{n−k}，E(X)=np，D(X)=np(1−p)
- **泊松分布**：X ∼ P(λ)，P(X=k) = λ^k e^{-λ}/k!，E(X)=λ，D(X)=λ

#### 3.2.4 常见连续分布

- **均匀分布**：X ∼ U(a,b)，f(x) = 1/(b−a)，E(X) = (a+b)/2，D(X) = (b−a)²/12
- **指数分布**：X ∼ Exp(λ)，f(x) = λe^{-λx} (x ≥ 0)，E(X) = 1/λ，D(X) = 1/λ²
- **正态分布**：X ∼ N(μ, σ²)，f(x) = (1/√(2πσ²))·exp(−(x−μ)²/(2σ²))
- **卡方分布**：χ² ∼ χ²(n)，n 个独立标准正态变量的平方和
- **Beta 分布**：定义在 [0,1] 上的连续分布，常用于贝叶斯统计中的先验分布

### 3.3 随机变量的数字特征

- **数学期望 E(X)**：平均水平
  - 离散型：E(X) = Σ xᵢ·pᵢ
  - 连续型：E(X) = ∫ x·f(x)dx
- **方差 D(X) 或 Var(X)**：波动大小，D(X) = E[(X−E(X))²] = E(X²) − [E(X)]²
- **协方差 Cov(X,Y)**：两个变量共同变化的程度，Cov(X,Y) = E[(X−E(X))(Y−E(Y))]
- **相关系数 ρ_XY**：标准化后的线性相关程度，ρ_XY = Cov(X,Y) / √(D(X)·D(Y))，|ρ| ≤ 1
- **矩**：k 阶原点矩 E(X^k)，k 阶中心矩 E[(X−E(X))^k]
- **协方差矩阵**：对多维随机向量 (X₁,...,Xₙ)，协方差矩阵为 Σ = [Cov(Xᵢ, Xⱼ)]

### 3.4 大数定律与中心极限定理

- **切比雪夫不等式**：P(|X−E(X)| ≥ ε) ≤ D(X)/ε²，给出了随机变量偏离其期望的概率上界
- **伯努利大数定律**：n 次独立重复试验中，事件 A 出现的频率依概率收敛于其概率
- **辛钦大数定律**：独立同分布的随机变量序列，其样本均值依概率收敛于数学期望
- **大数定律的意义**：样本平均会趋向稳定，为统计推断提供了理论基础
- **独立同分布中心极限定理（林德伯格-莱维）**：设 X₁,...,Xₙ 独立同分布，E(Xᵢ)=μ，D(Xᵢ)=σ²，则当 n 充分大时，ΣXᵢ 近似服从 N(nμ, nσ²)，即 (ΣXᵢ − nμ)/(√n·σ) ∼ N(0,1)
- **中心极限定理的意义**：大量独立随机变量之和（标准化后）趋于正态分布，这是统计推断、抽样理论和机器学习泛化分析的重要理论基础。

### 3.5 数理统计基本概念

- **总体**：研究对象的全体，通常用随机变量 X 表示
- **样本**：从总体中随机抽取的 n 个个体 X₁, ..., Xₙ，要求独立同分布（i.i.d.）
- **统计量**：样本的函数，不含未知参数，如样本均值 X̄ = (1/n)ΣXᵢ、样本方差 S²
- **三大抽样分布**：
  - χ² 分布：若 Xᵢ ∼ N(0,1) 独立，则 ΣXᵢ² ∼ χ²(n)
  - t 分布：若 Z ∼ N(0,1)，Y ∼ χ²(n) 且独立，则 T = Z/√(Y/n) ∼ t(n)
  - F 分布：若 U ∼ χ²(m)，V ∼ χ²(n) 独立，则 F = (U/m)/(V/n) ∼ F(m,n)

### 3.6 参数估计

#### 3.6.1 点估计

- **矩估计法**：用样本矩替换总体矩，解方程组得到参数估计值
- **最大似然估计（MLE）**：选择使样本出现概率最大的参数值作为估计值。即最大化似然函数 L(θ) = Π f(xᵢ; θ)，通常对 log L 求导
- **贝叶斯估计**：将参数视为随机变量，结合先验分布 π(θ) 和样本数据得到后验分布 π(θ|x)，以后验分布的均值或众数作为估计值
- **蒙特卡罗方法**：通过随机采样近似计算复杂积分和期望

#### 3.6.2 区间估计

- **置信区间**：对于参数 θ，区间 [θ̂_L, θ̂_U] 称为置信水平为 1−α 的置信区间，即 P(θ̂_L ≤ θ ≤ θ̂_U) = 1−α
- **构造方法**：利用枢轴量构造，如正态总体均值的区间估计

### 3.7 假设检验

- **基本概念**：
  - 原假设 H₀ 和备择假设 H₁
  - 显著性水平 α（第一类错误概率）
  - 检验统计量和拒绝域
- **Neyman-Pearson 基本引理**：在给定显著性水平下，似然比检验是最优的
- **Z 检验**：总体方差已知时检验均值，检验统计量 Z = (X̄ − μ₀)/(σ/√n) ∼ N(0,1)
- **T 检验**：总体方差未知时检验均值，检验统计量 T = (X̄ − μ₀)/(S/√n) ∼ t(n−1)
- **卡方检验**：检验总体方差或分类变量的独立性
- **应用**：假设检验常用于估计机器学习模型的泛化错误率，比较不同学习器的性能

### 3.8 回归分析

- **回归分析概述**：研究变量之间相关关系的统计方法，建立回归方程 y = f(x) + ε
- **一元线性回归**：y = β₀ + β₁x + ε
- **最小二乘法**：最小化残差平方和 RSS = Σ(yᵢ − β̂₀ − β̂₁xᵢ)²
  - 求解正规方程得到回归系数估计值
- **拟合优度 R²**：衡量回归方程对数据的解释程度，R² = 1 − RSS/TSS，取值范围 [0,1]，越接近 1 拟合越好
- **多元线性回归**：y = β₀ + β₁x₁ + ... + βₚxₚ + ε
- **曲线回归**：通过变量变换将非线性关系转化为线性回归
- **综合案例——个人医疗保费预测任务**：使用年龄、BMI、吸烟状况等特征预测个人医疗保费，通过回归分析建立预测模型

### 3.9 课程强调的理解方式


- 概率论解决“随机现象如何建模”。
- 数理统计解决“如何通过样本推断总体”。
- 这两部分共同支撑机器学习中的不确定性分析。

## 4. 数值分析

来源：`数值分析-Part1.pdf`、`数值分析-Part2.pdf`

### 4.1 函数空间与范数

#### 4.1.1 连续函数空间

- 连续函数空间是指所有连续函数的集合，在区间 [a,b] 上的连续函数空间记为 C[a,b]
- 函数空间中的元素是函数，可以进行加法和数乘运算

#### 4.1.2 常见范数

设 f ∈ C[a,b]，通常有三种范数：
- **∞-范数（最大值范数）**：||f||_∞ = max_{a ≤ x ≤ b} |f(x)|
- **1-范数**：||f||_1 = ∫_a^b |f(x)| dx
- **2-范数（均方范数）**：||f||_2 = √(∫_a^b |f(x)|² dx)

#### 4.1.3 范数的作用

- 用于衡量误差大小和函数的接近程度
- 用于分析插值、拟合和逼近的收敛性

### 4.2 函数插值

#### 4.2.1 插值的基本思想

- 已知若干节点 (x_i, y_i) 及函数值，构造一个函数 L(x) 使其通过这些节点，即 L(x_i) = y_i
- 插值在图像处理中的应用：对图像进行放大或缩小，插入像素后给这些像素赋值，从而恢复图像内容

#### 4.2.2 常见方法

- **拉格朗日插值**：L_n(x) = Σ_{i=0}^{n} y_i · l_i(x)，其中 l_i(x) = Π_{j≠i} (x−x_j)/(x_i−x_j) 为拉格朗日基函数
  - n 次插值多项式唯一存在，误差项与 f^{(n+1)}(ξ) 成正比
- **牛顿插值**：利用差商（divided difference）递推构造，便于增加节点
  - N_n(x) = f[x₀] + f[x₀,x₁](x−x₀) + ... + f[x₀,...,x_n](x−x₀)...(x−x_{n−1})
- **分段插值**：将区间分为若干段，每段用低次多项式插值，避免高次插值的振荡（龙格现象）
- **样条插值**：分段光滑多项式插值，常用三次样条（Cubic Spline），保证二阶导数连续
- **径向基函数（RBF）插值**：以径向函数为基的插值方法，适用于高维散乱数据插值

#### 4.2.3 插值与误差

- 插值多项式次数升高不一定更好（龙格现象 Runge Phenomenon）
- 节点分布与误差控制非常关键，切比雪夫节点分布可缓解高次插值的误差
- 插值余项（误差）：R_n(x) = f^{(n+1)}(ξ)/(n+1)! · Π (x−x_i)

### 4.3 函数拟合


#### 4.3.1 拟合的基本思想


- 拟合不要求通过全部数据点，而是寻找“整体上最好”的模型。

#### 4.3.2 课程中的核心概念


- 训练集
- 验证集
- 测试集
- 泛化误差
- 经验误差

#### 4.3.3 欠拟合与过拟合


- 欠拟合：模型太弱，抓不住数据规律。
- 过拟合：模型太强，把噪声也当成规律。
- 课程明确指出：二者都会损害泛化能力。

#### 4.3.4 偏差-方差分解


- **偏差（Bias）**：所有可能的训练数据集训练出的模型的平均输出与真实输出的差异。偏差高说明模型偏简单
  - bias^2(x) = f_bar(x) - y，度量期望预测与真实结果的偏离程度
- **方差（Variance）**：不同训练集训练出的模型输出值之间的差异。方差高说明模型对数据扰动太敏感
  - var(x) = E_D[(f(x;D) - f_bar(x))^2]
- **偏差-方差权衡**：简单模型偏差高方差低，复杂模型偏差低方差高

#### 4.3.5 正则化方法


- **L2 正则化（Ridge Regression）**：在损失函数中加入权重向量的 L2 范数平方，防止过拟合
  - 作用：限制模型空间，使模型更简单；有助于病态矩阵求逆
  - 概率解释：相当于参数服从高斯先验分布
- **L1 正则化（Lasso）**：加入权重向量的 L1 范数，产生稀疏解
  - 与 L0 关系：L1 是最优凸近似
  - 概率解释：相当于参数服从拉普拉斯分布
- **L1 vs L2 对比**：L1 趋向特征选择（稀疏），L2 让特征接近 0 但不为 0

### 4.4 函数逼近

- **函数逼近问题**：用简单的函数 g(x) 近似代替复杂的原函数 f(x)，且误差可以量化
- **最佳逼近函数的定义**：设 Φ ⊂ C[a,b] 为某个函数空间，若存在 g*(x) ∈ Φ，使得 ||f − g*|| = min_{g∈Φ} ||f − g||，则称 g*(x) 为 f(x) 在 Φ 中的最佳逼近函数
- **最佳一致逼近（切比雪夫逼近）**：使用 ∞-范数衡量误差，使最大绝对误差最小化
- **最佳平方逼近**：使用 2-范数衡量误差，使平方误差的积分最小化
- 逼近思想是数值计算、信号处理和机器学习的重要基础。

### 4.5 与机器学习的联系


- 数值分析中的“拟合”与机器学习中的“训练”本质相关。
- 课程重点把“模型拟合”放到泛化框架里理解，而不是只看训练集表现。

## 5. 图论

来源：`图论.pdf`

### 5.1 图论的基本概念

- 图由顶点（Vertex）和边（Edge）构成，图论研究事物之间的关系结构
- **图的分类**：
  - 有向图 vs 无向图：边是否有方向
  - 简单图 vs 多重图：是否有平行边和自环
  - 完全图：任意两个不同顶点之间都有边相连
  - 加权图：边上有权值
- **哥尼斯堡七桥问题**：1736 年欧拉发表《柯尼斯堡的七桥》，是图论的第一篇文章。欧拉将实际问题抽象为图，证明不存在经过每座桥恰好一次的路径（欧拉回路）
- **四色问题**：任何地图都可以用四种颜色着色，使相邻区域颜色不同
- **汉密尔顿问题**：寻找经过每个顶点恰好一次的回路（汉密尔顿回路）

### 5.2 图论的发展背景

- 萌芽阶段（18 世纪中叶到 19 世纪中叶）：围绕游戏产生，从七桥问题开始
- 发展阶段（19 世纪中叶到 20 世纪中叶）：四色问题、汉密尔顿问题、树在化学中的应用、基尔霍夫用树分析电力网络、1936 年哥尼格的《有限图与无限图理论》
- 应用阶段（20 世纪中叶至今）：由于生产管理、军事、计算机网络、数字通讯等需求，特别是 Ford 和 Fulkerson 的网络流理论，图论得到了飞速发展

### 5.3 基本术语

- **顶点（Vertex）**：图的基本元素
- **边（Edge）**：连接两个顶点的线
- **度（Degree）**：与顶点相连的边的数目
- **路径（Path）**：顶点和边的交替序列
- **回路（Cycle）**：起点和终点相同的路径
- **连通图**：任意两个顶点之间都存在路径的图
- **树（Tree）**：无回路的连通图，n 个顶点的树有 n−1 条边
- **最小生成树（MST）**：连通加权图中边权和最小的生成树
- **子图**：从一个图中取出一部分顶点和边构成的新图
- **图的遍历算法**：
  - 深度优先搜索（DFS）：沿着一条路径走到尽头再回溯，用栈实现
  - 广度优先搜索（BFS）：从起点开始逐层向外扩展，用队列实现

### 5.4 谱图理论

- 图可以用矩阵表示：
  - **邻接矩阵 A**：A_ij = 1 若顶点 i 和 j 相邻，否则为 0（无向图为对称矩阵）
  - **度矩阵 D**：对角矩阵，D_ii = deg(v_i)
  - **拉普拉斯矩阵 L**：L = D − A，是半正定矩阵，特征值 0 的代数重数等于连通分量数
  - **关联矩阵**：行对应顶点、列对应边，描述顶点与边的关联关系
- 图的结构信息可以通过矩阵特征值分析
- **谱图理论与谱聚类**：利用拉普拉斯矩阵的第二个最小特征值（Fiedler 值）对应的特征向量进行图的二分分割，是谱聚类的核心思想

### 5.5 谱图理论的意义


- 与聚类、图划分、网络分析密切相关。
- 是图论与线性代数结合最紧密的部分之一。

### 5.6 图论与 AI 的联系


- 社交网络、知识图谱、推荐系统、图神经网络都离不开图结构。
- 谱聚类就是图论与机器学习结合的典型例子。

## 6. 信息论

来源：`信息论.pdf`

### 6.1 信息论概述

- **克劳德·香农**：1948 年发表《通讯的数学理论》，创立信息论。香农也是密码学家，与图灵有过交流
- **信息论的历史发展**：
  - 1928 年哈特莱发表《信息传输》，认为"信息是选择的自由度"
  - 1948 年香农创立信息论，认为"信息是事物运动状态或存在方式的不确定的描述"
  - 1950 年维纳提出"信息既不是物质又不是能量，信息就是信息"
  - 20 世纪 70 年代后：广义信息论（信息科学）逐渐形成
- **信息的特征**：广泛性（客观世界充满信息）、抽象性（信息是独立于物质和能量的第三要素）
- **信息的分类**：语法/语义/语用信息，连续/离散信号，前馈/主馈等
- **通信系统基本模型**：
  信源 → 编码器（信源编码 + 信道编码）→ 信道（引入噪声）→ 译码器（信道译码 + 信源译码）→ 信宿
- **通信的三项性能指标**：
  - 有效性（有效性）：数据压缩、高传信率
  - 可靠性（可靠性）：低误码率、纠错编码
  - 安全性（安全性）：数据加密、防止泄露

### 6.2 核心概念

#### 6.2.1 自信息与信息熵

- **自信息量**：I(x) = −log₂ P(x)，度量单一事件的信息量，单位比特（bit）
- **条件自信息量**：I(x|y) = −log₂ P(x|y)
- **联合自信息量**：I(x,y) = −log₂ P(x,y)
- **信息熵（平均自信息量）**：H(X) = −Σ P(x) log₂ P(x)，衡量随机变量 X 的不确定性
  - 分布越平均，熵越大；确定性分布熵为 0
  - 熵函数的性质：对称性、非负性、极值性（均匀分布时熵最大）

#### 6.2.2 条件熵与联合熵

- **条件熵**：H(Y|X) = Σ P(x) H(Y|X=x) = −Σ P(x,y) log₂ P(y|x)
  - 已知 X 后 Y 还剩多少不确定性
- **联合熵**：H(X,Y) = −Σ P(x,y) log₂ P(x,y)
  - 关系：H(X,Y) = H(X) + H(Y|X) = H(Y) + H(X|Y)

#### 6.2.3 互信息

- **互信息量**：I(X;Y) = Σ P(x,y) log₂ [P(x,y) / (P(x)P(y))]
- 描述两个随机变量之间共享了多少信息
- I(X;Y) = H(X) − H(X|Y) = H(Y) − H(Y|X) = H(X) + H(Y) − H(X,Y) ≥ 0
- **条件互信息量**：I(X;Y|Z) = H(X|Z) − H(X|Y,Z)

#### 6.2.4 KL 散度与相对熵

- **相对熵（KL 散度）**：D_KL(P||Q) = Σ P(x) log₂ [P(x)/Q(x)]
  - 衡量两个分布 P 和 Q 之间的差异
  - D_KL(P||Q) ≥ 0，等号当且仅当 P = Q
  - 不满足对称性和三角不等式（不是真正的距离）
- 在机器学习中，交叉熵损失函数等价于最小化 KL 散度

### 6.3 信息论与 AI

- **信息熵**在决策树中的应用：选择信息增益最大的特征进行分裂
- **互信息**用于特征选择和表示学习
- **编码与压缩思想**与模型复杂度控制相关（最小描述长度 MDL 原则）
- 交叉熵损失函数：通过最小化预测分布与真实分布的 KL 散度，使预测值变量的分布接近真实值变量的分布
