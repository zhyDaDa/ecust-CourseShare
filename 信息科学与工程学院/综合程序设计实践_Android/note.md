# 综合程序设计实训_Android

我愿称这门课为: **完全AI开发产品实践**  

没有任何的教学, 你要做的就是:

1. 安装 android studio
2. 按照他教的去开一个项目
3. 在VS code里面, 搞一个AI的插件
   > 该充钱的时候就充, 如果搞了老半天，发现最后还是得充，那就得不偿失了
   > 1. 最便宜的是codeGeex是免费的, 没法做大的效果。
   > 2. 淘宝买一个月的Copilot Pro
   > 3. 买deepseek或者kimi的API, 接 CC Switch, 接到Codex/Claude
   > 4. 直接买官方GPT
4. 你要在那个项目文件夹里面找到最主要的那段源代码，这个你可以问AI
5. 在这个源代码的路径下，打开VS code，让它能够访问到全部的核心代码
6. 在vscode里面开一个git
7. 然后你让AI去按照你的想法写, 写完了之后回到android studio去编译验证。
8. 每做好一小块就保存commit

相信我, 这可能就是最好的方案了

还有一些注意事项:

1. 你一定要做全套的功能，不要偷懒, 学长会挑刺。
2. 要有一个比较独特的功能, 我做的功能就是内嵌一个web浏览器去访问我做的游戏。
3. 记得多commit, 最后学长会检查的。
4. 这个应用还是比较全面的，它里面涉及到了数据库，你要注意的是很多时候突然这个程序怎么样都运行不了了，这个时候很可能是因为你数据库版本不一样了。你之前的代码当中表头和现在代码的表头不一样，它访问数据库就会失败，这个时候你要把前面的数据库删掉，再重建数据库才可以。
