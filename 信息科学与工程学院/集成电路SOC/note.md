# 集成电路SOC

相当抽象的一门课，其实你最主要的就是学会在Linux的终端中做操作。  
因为soc开发只能依赖这个软件，上课的内容就是教你首先用虚拟机把这个系统跑起来，然后就是教你怎么在Linux中用vim来操作文件和一些基本的指令。  
然后就是按照他实验要求来，要跑一个程序，要能编译出一个结果  
我真的谢天谢地没要我们去设计什么东西出来, 整个过程一直是云里雾里的  
即便你照做都不一定能跑得起来
